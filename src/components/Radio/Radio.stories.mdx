# Radio

A radio button typically represents a single option in a mutually exclusive list (where only one item can be selected at a time). We recommend to use with `RadioGroup` component.

Radio buttons should:

- Always be used with an associated label component.
- Be part of a list of radio buttons that:
- Include at least two or more choices.
- Are used to have users select only one option.


## Import  

```jsx
import { Radio } from '@stack-ui/components';
```
